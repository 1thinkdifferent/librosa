librosa.filters.constant_q_lengths¶

librosa.filters.constant_q_lengths(sr, fmin, n_bins=84, bins_per_octave=12, tuning=0.0, window='hann', filter_scale=1, resolution=<DEPRECATED parameter>)[source]¶

Return length of each filter in a constant-Q basis.

Parameters:

sr : number > 0 [scalar]

Audio sampling rate

fmin : float > 0 [scalar]

Minimum frequency bin.

n_bins : int > 0 [scalar]

Number of frequencies. Defaults to 7 octaves (84 bins).

bins_per_octave : int > 0 [scalar]

Number of bins per octave

tuning : float in [-0.5, +0.5) [scalar]

Tuning deviation from A440 in fractions of a bin

window : str or callable

Window function to use on filters

filter_scale : float > 0 [scalar]

Resolution of filter windows. Larger values use longer windows.

resolution : float

Warning

This parameter name was in librosa 0.4.2 Use the filter_scale parameter instead. The resolution parameter will be removed in librosa 0.5.0.
Returns:

lengths : np.ndarray

The length of each filter.

See also

constant_q, librosa.core.cqt
